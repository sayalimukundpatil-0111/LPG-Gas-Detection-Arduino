# LPG Leakage Detection System using Arduino

## Description
This project detects LPG leakage using an MQ-2 sensor and alerts users using a buzzer and LED and simultaneously cutoff further gas flow with the help of servo motor.

## Components Used
- Arduino Uno
- MQ-2 Gas Sensor
- Buzzer
- LED
- Servo Motor
- Jumper wires

## Working
The MQ-2 sensor senses LPG gas. When gas level exceeds a threshold, Arduino activates buzzer, LED to alert and activate rotation of servo motor to prevent further gas flow.

## Applications
- Home safety
- Industrial gas monitoring

## Result
The system successfully detects gas leakage and provides an alert.
