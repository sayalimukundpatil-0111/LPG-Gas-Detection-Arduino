# LPG Gas Leakage Detection System using Arduino

## Description
This project detects LPG gas leakage using an MQ-2 sensor and alerts users using a buzzer and LED.

## Components Used
- Arduino Uno
- MQ-2 Gas Sensor
- Buzzer
- LED
- Jumper wires

## Working
The MQ-2 sensor senses LPG gas. When gas level exceeds a threshold, Arduino activates buzzer and LED to alert.

## Applications
- Home safety
- Industrial gas monitoring

## Result
The system successfully detects gas leakage and provides an alert.
